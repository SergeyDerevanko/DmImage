#DmImage

[http://demouth.github.com/DmImage/](http://demouth.github.com/DmImage/)

###DmImageについて
* PHPライブラリ
* PHP5.0以上で動作
* GDのラッパー
* GDが入っていれば動く
* 対応するフォーマット
  * jpg(jpeg)
  * png
  * gif
* 注意点
  * 円の描画の際、線を半透明にするとあまり綺麗に描画されない
  * bmpに対応していない
